What's an AMYboard?

It's fully complete audio synthesizer for US$29.90 that can be at least three things:

🎹 Synthesizer

A full-featured, ready-to-play synthesizer module with lush 80s polyphonic synthesizer emulation. Load shared patches and presets and tweak your synth online.

🛠️ Dev Board

A maxed-out audio development board with full analog and digital IO, support under Arduino and native dev tools, and an out-of-the-box on-device Micropython shell.

🎛️ Eurorack

A 10HP Eurorack module that can be reprogrammed for almost any conceivable modular synth function from oscillators, filters, and envelopes to nonlinear effects and CV transformations. It's a tenth of the price of similar modules. Get two!

What it can do

Try out some examples on AMYboard online, and send them to your AMYboard in seconds!

Generative Acid patch (303 + 808) We set up a TB303 and 808 and play a generative pattern. Twiddle the knobs!! A custom 4-op FM synth We set up a custom ALGO patch in Python modeled after the infamous WOOD PIANO preset. Play a MIDI file We load a MIDI file from disk and play it with a synth patch. A spacey synth patch Global effects (echo, reverb) on top of a slow envelope over PWM. Control your filters with CV input We use the AMY CtrlCoef EXT0 to have CV input modulate the filter frequency. Convert between SPDIF and analog audio Set up AMY to pass AUDIO_IN to AUDIO_OUT so that S/PDIF outputs over analog line-out, and vice versa. MIDI controller to CV We set up a callback to get MIDI controller data and output scaled CV values. DX7 setup with filter and reverb Try setting the MIDI CC for VCF freq! Arpeggiator Uses MIDI input callbacks and the AMY sequencer to set up an arpeggiator Echo / reverb on live audio Shows how to use AUDIO_IN to add effects on live audio. Turn on "allow audio input" on the web and move the effects! Chord computer Plays chords based on a root note. Load a wav file into a sampler Loads a WAVE file from disk (can also be SD card) and plays it back as a sampler. Xanadu Joseph Kung's "Xanadu" for Csound, ported to AMY commands in code. Wavetable patch Uses our WAVETABLE oscillator type to play wavetable synthesis. Patch Selector on OLED Our built in patch selector, use the rotary encoder to select different saved patches, press to select. Preset Selector Sets up the rotary encoder to cycle through all 257 AMYboard presets. Additive-synthesis piano Uses the AMY dpwe piano patch, fully synthesized from partial analysis. Generative House A generative house track with 808 drums, Wood Piano bassline, and piano chords over evolving progressions. Evolving Chords Lush evolving chord progressions with layered synths and generative voice leading.
